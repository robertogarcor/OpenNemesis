# Habilidad de Ejemplo

Esta habilidad sirve como referencia visual para otros desarrolladores.

## Cuándo usar esta habilidad
- Para ver el formato correcto de un archivo `SKILL.md`.

## Cómo usarla
1. Observa el frontmatter arriba.
2. Sigue los encabezados de nivel 1 y 2.
3. Asegúrate de que la descripción sea clara para que el agente la encuentre.
